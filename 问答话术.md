
本文整理了一些问答话术，用于聊骚和谈恋爱。

# 聊骚话术
聊骚话术主要适用于暧昧对象，有的话太有聊骚性质，注意区分使用场合。

对有些稳重的女孩子直接使用会有点不太礼貌。

# 情景1

男：我们在一起吧。

女：我们发展太快了，我们还需要再了解了解。

分析：出现这种情况，是女生没有安全感的表现，怕太快得到不够珍惜，顺带借机会考验，看看追求的决心。

错误回答
男：你还想了解我什么。

正确回答：

男：宝贝，你如果看上一个非常喜欢的包包，你给他买下来，你觉得快吗？

女：不快。

男：感情没有快慢，只有深浅。有的人相识很久缺形同陌路，而你我相识却一眼万年。


# 情景2
女：你为什么对我这么好？

错误回答：
男：因为我喜欢你呀。

分析：莫名其妙告白，瞬间变成舔狗，把主动权交给了女生，以后只能被吊着。女生明明知道你对他的好，他真正想问的是后面没有问出来的话，你是不是喜欢我。如果直接顺着说，就进入了她的圈套。
女生想要通过揭露男生的心思，引导男生向他表白。所以我们应该揭露女生的心思，然后把我主动权。

正确：听说，喜欢一个人，就会无限放大他对你的好。

# 情景3

女：我要睡了，晚安。

错误回答：

男：好的，晚安。

这种回答就没有了后续，

建议回答：

男：其实...

女：其实什么？

男：其实相比晚安，我更加想睡醒以后亲口对你说早安。

不管后续女生回答啥，一律回答，安。然后去睡觉，不用理她，让她心动一整晚。

# 情景4

女：早安。

分析：潜台词，老娘睡醒了，你可以来撩我了。

错误回答：

男：早安，你吃饭了吗。

建议回答：

睁眼就能看到你这消息真好，要是哪天一睁眼就能看到你那更好了。

分析：这种方式，反客为主，顺手牵羊。这样即表达了真心，也描绘了你们在一起的美好画面。

# 情景5

女：我去洗澡啦。

错误回答：

男：好的。

分析：暧昧的话语，注意是"啦"不是"了"，这是女生想要撩你。我们应该返撩回去。

建议回答：

男：我在你厕所里装了摄像头。

女：你不怕摄像头被水浇坏吗？

男：那你要赔我摄像头的钱哦。

# 情景6

女：你谈过几段恋爱？

错误回答：

男：第一段，xxx，第二段，xxx。

分析：如实回答，回答的多了，显得海王，显得念念不忘。回答的少了，说明自己没有吸引力。是一张白纸，还要去教。
女生问这个说明是想了解你的过去，并且对你有兴趣。所以我们要回答这个问题，并且也要调动他的情绪。

建议回答：

男：我谈过两次，一次是学生时期的懵懂无知，第二次让我学会了如何更好的经营一段感情。第三次，我希望可以白头偕老。

# 情景7

女：你喜欢怎么样的女生？

错误：

男：我喜欢xxx，温柔、体贴的女生。

分析：女生听完以后，就觉得自己非常复合标准，就没有其他的什么感觉了。这时候就觉得老娘吃定你了。
女生会用你喜欢的样子，吸引你，让你成为她的备胎。

建议回答：

男：还挺多人问我这个问题的。

女：那你平时都是怎么回答的？

男：我都是回答，喜欢温柔、自律、懂得分寸的、懂得回应的女生。不过每当这时候，我都是形容你。

# 情景8

女：你是不是想撩我？

错误：

男：我不是在撩你，我是在撩我未来的老婆。

分析：这种回答太油腻，过早的暴露了自己的需求感，暴露了自己对他的兴趣，这样自己就失去了神秘感。
可以使用公式：泼冷水＋嘲讽＋爱称＋比喻。

建议回答：

呵呵，你想多了，你是太久没有男生追了吧，老修女，你是不是吃了鸭舌都有舌吻的感觉啊。

个人看法：

感觉好像有点太过攻击性，不太合适。

# 情景9

女：.....(女生总发省略号)

错误：

男：你为啥总会发.....，你不会说话吗？

建议回答：

男：你不知道我是研究标点符号的嘛？

女：你研究出了什么了嘛？

男：听说发标点符号的人，不是因为他敷衍你，而是因为他什么也说不出。可是他还在。你觉得对吗？

个人看法：

很尴尬，有毛病



# 恋爱话术
恋爱话术主要适用于女朋友。

# 情景1

女：你为什么喜欢我。

错误回答：

因为你很好呀，你很善良，等等夸赞的话。

分析：这是一种没有安全感的表现。傻逼下意识的就会给出一个夸赞的话。这时候女生心里回想，以后如果也遇到一个这样的人，是不是就不会喜欢自己了。其实女生问的还有后半句，你为什么喜欢我(而不是喜欢别人）。可能是男生周围出现了新的女生，让女生觉得自己原来非常普通。
所以不能只夸女生的优点。因为具有一样优点的人很容易被替代。应该使用"情不知所起、一往而情深。"要肯定女生的不可替代性。
哪怕有一个再优秀的人也不能替代她。

建议回答：

曾经的我以为，我喜欢的是你的优秀，现在发现啊，这些都不重要，我喜欢你没有理由，但是你是我不喜欢别人的理由。
